Catch the Point

Try to catch the red point!

How to Play

Controls:

  • Use Arrow Keys to move the Blue Point
  • WASD Keys for Red Point (Two-Player Mode only)

Game Modes:

  • Offense Mode: You (Blue) try to catch Red. Red moves first and tries to escape!
  • Defense Mode: You (Blue) try to escape! Red moves first and tries to catch you!
  • Two-Player Mode: Red uses WASD, Blue uses Arrows

Rules:

  • After each move, a random path disappears
  • Game ends when points connect or become separated
