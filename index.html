Catch the Point

Try to catch the red point!

How to Play

Controls:

  • Blue Point (Player 1): Arrow Keys
  • Red Point (Player 2): WASD Keys (Two-Player Mode)

Game Modes:

  • Offense Mode: Try to catch the red point! Red moves first and tries to escape.
  • Defense Mode: Try to escape from the red point! You move first, then red chases.
  • Two-Player Mode: Red uses WASD, Blue uses Arrows

Rules:

  • After each move, a random path disappears
  • Game ends when points connect or become separated

Strategy Tip: Plan your moves carefully! The disappearing edges can either help or hinder your progress.
